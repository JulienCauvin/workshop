# PHP Workshop

## Prerequisites

- php 7.4
- mysql

## Usage

Run script :
```bash
cd path/to/script/
php my-script.php <args>
```

Start development server :
```bash
cd path/to/folder/to/serve/
php -S localhost:9000
```

Import MySql dump :
```bash
cd ./3-database/
mysql -u <user> -p <database_name> < dump.sql
```
